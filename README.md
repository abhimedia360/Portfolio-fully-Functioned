**# Portfolio – Fully Functioned (HTML/CSS/JS)

A simple multi-page personal portfolio built with **HTML**, **CSS**, and **JavaScript**. It includes a landing page, a portfolio/projects page, and a contact page with basic interactivity.

> Repository: https://github.com/abhimedia360/Portfolio-fully-Functioned

---

## Overview

This project is a lightweight, front-end-only portfolio site you can open directly in a browser or host on GitHub Pages. It’s easy to customize—just edit the HTML, tweak the CSS, and add your own images and content.

### Pages & Files

- `index.html` — Landing/Home page  
- `PORTFOLIO.HTML` — Portfolio/Projects page  
- `contact.html` — Contact page  
- `home.js`, `contact.js` — Small JS for navigation/form and interactivity  
- `style.css`, `styleport.css`, `home.css`, `contact.css` — Page-specific styles  
- Image assets — e.g. `HOMRBACKGROUND2.png`, `bg.jpg`, `homebackground.jpg`, `housebg.jpg`

> Tip: The “View Portfolio” button on the home page should link to `PORTFOLIO.HTML`. Update the button’s `href` if you rename files.
